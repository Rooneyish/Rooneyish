### 🚀 Core Engineering Projects

* 🤖 [**Hansei**](https://github.com/Rooneyish/Hansei) — An AI-driven cognitive well-being tracking platform engineered with a high-performance **FastAPI** backend, **Node.js**, **React Native** for mobile, and semantic RAG vector searches utilizing **Qdrant**.
* 📝 [**Devanagari Character Recognition System**](https://github.com/Rooneyish/devanagari_ocr_app) — A custom deep learning pipeline using CNN and MobileNetV2 architectures to seamlessly transcribe and recognize handwritten script.
* 🛰️ [**Deforestation Detection**](https://github.com/Rooneyish/deforestation_detection) — A computer vision framework leveraging remote sensing satellite imagery and semantic segmentation models to detect, track, and monitor environmental canopy loss over time.
* 🎭 [**GoEmotion Classifier**](https://github.com/Rooneyish/GoEmotion_Classification) — A fine-tuned Natural Language Processing (NLP) pipeline optimized for granular multi-label emotion classification, mapping text streams to complex psychological states.
* 🎮 [**GameSpace**](https://github.com/Rooneyish/GameSpace) — A scalable standalone Java application engineered to structure and manage multi-layered game information records.
